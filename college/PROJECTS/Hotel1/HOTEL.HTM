This tutorial help offers a GUIDED TOUR to the features of the database you might find slightly obscure.

The main form consists of six major menu options. They are:

  • Open Menu

  • View Menu

  • Find

  • Help

  • Exit

  1. The OPEN Menu is a drop down menu which allows you TO ENTER SPECIFIC DETAILS OF THE FOLLOWING:

  • CUSTOMER INFORMATION: The option displays the "CUSTOMER INFORMATION" form which allows you to view, add, delete, and modify the details of the customers checking-in and checking-out of the hotel.

  • ROOM INFORMATION: This option allows you to view, add and even modify the room information by way of the room number, its current occupancy status, and the type of room, i.e the facilities it has to offer.

  • TARIFF DETAILS: This option allows you to view the tariff details of each specific room. Please note that the fares are charged in Rs. per DAY.

  • EMPLOYEE DETAILS: This option allows you to view the hotel's employee database. It gives a detailed information of the employees, their residential address, their respective departments and their designation and salaries.

  • DEPARTMENT INFORMATION: This option allows you to specifically view the various departments in the hotel, along with adding new or deleting the unrequired departments.

  • CREATE BILL: This option allows you to create the bill informaton fot the customer, along with allowing you to view previous bills, an make changes, if any.

  2.   VIEW: This menu option provides you with a TABULAR VIEW of the following:

  • Customer Details

  • Department Information

  • Employee Details

  • Room Information

  • Tariff Details

  • Bills

  3.  FIND: This menu option allows you to make a QUERY SPECIFIC SEARCH in the database for finding a particular field entry. CHOOSE the field as per your wish for making the search and the query details. Be sure of the type you enter, else the search is bound to give error messages !!

  4. HELP: That's where you  are right now !! Yes, the detailed help manager helps you through the database features with simplicity and ease.

 5.  EXIT : This option allows you to EXIT the database.
